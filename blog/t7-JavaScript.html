JavaScript Objects and Ruby Hashes

Aug 1, 2015

JavaScript Objects and Ruby Hashes definitely have some similarities but are different. Hashes in Ruby are a key-value pair while in JavaScript they are a property-value pair. Declaring a Ruby Hash can be as simple as the following:

hash = {}

Similarly, in JavaScript creating an object is done like this:

var object = {};

Except for the keyword var and semicolon they have very similar syntax.

Adding to each can be similar too. In Ruby:

hash["key1"] = "value1"

and in Javascript it is

object["property1"] = "value1";

But in JavaScript you can also add by dot notation like so:

object.property1 = "value1";

The major differences are of syntax and naming at a superficial level.
