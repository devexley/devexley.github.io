Enumerables

The Map Method

July 11, 2015

Ruby has an enumerable method called "map" which allows you to transform values iterating over an array. It takes a block and executes it for each element of an enumerable object such as an array. The object therefore has to have an "each" method to utilize "map". There is the non-destructive map which outputs a new array and a destructive map! which updates the original array.

Here is an example syntax of the non-destructive map function which takes the original array list = [1,2,3,4,5] and returns the square of each in a new array.

list.map {|i| i ** 2} => [1, 4, 9, 16, 25] list => [1, 2, 3, 4, 5]

Notice we get the square return in a new array but the original array of list still contains the original values. map! however is a destructive method and alters the original list as shown below.

list.map! {|i| i ** 2} => [1, 4, 9, 16, 25] list => [1, 4, 9, 16, 25]

This is very useful for executing a block on each element in an array to perform an operation according to the block expression because you can produce a new array preserving the original or update the original.
